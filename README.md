# Morgames

Morgames is a gaming ecosystem where anyone can create, play, and share interactive experiences.

Using a visual Block Game Builder, users can transform ideas into playable games through a simple and accessible creation system.

No complex code.
Just creativity, logic, and imagination.


## About

Morgames is a platform focused on game creation and discovery.

It combines visual development tools, a creator dashboard, and community features into one ecosystem.

The goal is to make game development accessible to everyone, allowing players to become creators.


## Features

### Block Game Builder

Create games using a visual block-based system.

- Custom logic
- Dynamic events
- Game mechanics
- Rapid prototyping


### Creator Dashboard

Manage your projects and gaming experience.

- Project management
- Statistics
- Progress tracking
- Game analytics


### Community

Share creations and discover games from other creators.

- Game publishing
- Creator profiles
- Ratings and feedback
- Leaderboards


## Vision

Morgames aims to become a platform for the next generation of game creators.

A place where ideas become games and creativity is not limited by technical knowledge.
