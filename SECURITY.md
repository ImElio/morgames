# Security Policy

## Reporting a Vulnerability

If you discover a security vulnerability in Morgames, please do not publicly disclose it immediately.

Report it privately with:

- A description of the vulnerability.
- Steps to reproduce the issue.
- Potential impact.
- Any suggested solutions.


## Responsible Disclosure

We ask security researchers to:

- Avoid exploiting vulnerabilities.
- Avoid accessing user data.
- Allow reasonable time for a fix before public disclosure.


## Scope

This policy applies to:

- Morgames source code.
- Official Morgames services.
- Infrastructure controlled by the project.


Thank you for helping keep Morgames secure.
